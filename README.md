## Requirement

- Node.js 6.11.0
- Ionic CLI 3.4.0


## Installation

    $ git clone https://github.com/adash333/ionic3-calculate.git
    $ cd ionic3-timer
    $ npm install
    $ ionic serve

androidアプリの作成(ionic-cli version3.xの場合、通常はこちら)

    $ ionic cordova platform add android
    $ ionic cordova build android
